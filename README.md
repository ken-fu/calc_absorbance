calc absorbance
==================
  
## Overview
This program calculates the absorption spectrum (Absorbance) based on Lambert-Beer's law.
  

## Requirement
python 3.6.2  
matplotlib==2.0.2  
numpy==1.13.1  
PyQt==5.12.2

## Usage
Drag and drop the required data (TSV or CSV) and press calc.
